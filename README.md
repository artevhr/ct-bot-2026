# 🎓 ЦТ-Бот 2026 — Беларусь

Телеграм-бот для подготовки к ЦТ 2026 по трём предметам:
**📝 Русский язык · 📐 Математика · 🇬🇧 Английский язык**

---

## 🚀 Быстрый старт

### 1. Клонируй репозиторий
```bash
git clone https://github.com/твой_ник/ct-bot-2026.git
cd ct-bot-2026
```

### 2. Установи зависимости
```bash
pip install -r requirements.txt
```

### 3. Создай .env
```bash
cp .env.example .env
# Открой .env и заполни BOT_TOKEN и ADMIN_IDS
```

### 4. (Необязательно) Добавь PDF сборники
Скопируй PDF файлы в папку `media/books/`:
- `rus_60_urokov.pdf`
- `rus_tablicy.pdf`
- `eng_karnevskaya.pdf`
- `math_larchenko.pdf`
- и другие (см. data/collections.py)

### 5. Запусти
```bash
python bot.py
```

---

## 🚂 Деплой на Railway

1. Зайди на [railway.app](https://railway.app)
2. New Project → Deploy from GitHub → выбери репозиторий
3. Добавь переменные в **Variables**:
   - `BOT_TOKEN` — токен от @BotFather
   - `ADMIN_IDS` — твой Telegram ID (узнай у @userinfobot)
   - `WEBHOOK_URL` — URL Railway-проекта (появится после первого деплоя)
4. Railway подхватит `railway.json` автоматически

> ⚠️ `WEBHOOK_URL` — добавить ПОСЛЕ первого деплоя. 
> Формат: `https://ct-bot.up.railway.app` (без слеша в конце)

## ✨ Функции бота

**📝 Тест-система:**
- РТ 2026: Русский Этап III В2 (40 заданий А1–А18 + В1–В22)
- РТ прошлых лет: 2022–2025 (из архивов РИКЗ)
- По каждому заданию — кнопка 📖 Теория
- После теории — кнопка вернуться к тому же заданию (прогресс не сбрасывается)
- Пропустить вопрос → вернуться позже
- Итог: первичный балл → тестовый балл + разбор каждого вопроса

**📖 Теория:**
- Русский: 29 тем (орфография, пунктуация, фонетика, лексика, морфология)
- Английский: 13 тем (грамматика, лексика, шпаргалки)
- Математика: 15 тем (алгебра, геометрия, текстовые задачи)
- Шпаргалки по каждому предмету

**✏️ Доп. задания:**
- Английский: словообразование (3 варианта), модальные глаголы (2 варианта)

**📚 Сборники:**
- Отправляет PDF файлы сборников прямо в чат

**🧮 Калькулятор баллов:**
- Таблицы перевода РИКЗ для всех 3 предметов

**👤 Личный кабинет:**
- Тестов решено, лучший балл, статистика по предметам

**🛠 Админ-панель:**
- Количество пользователей, статистика по тестам, рассылка

## ➕ Добавить новый тест

В `data/questions.py` в каталог `TESTS_CATALOG`:

```python
catalog["russian"]["2026_э1_в1"] = {
    "title": "РТ 2026 — Этап I, Вариант 1",
    "questions": [...],      # список заданий
    "max_primary": 80,
    "subject": "russian",
    "is_2026": True,
}
```

Структура задания:
```python
{
    "id": "А1",
    "type": "A_multi",           # A_single | A_multi | B_text | B_match
    "text": "Текст задания",
    "answer": "1,3,5",           # для A: "1,3"; для B: "слово"
    "theory_key": "орф_1",       # ключ в data/theory_*.py
    "theory_hint": "Тема",       # текст кнопки
    "points": 2,
    "partial": True,             # возможен частичный балл
}
```
